使い方（概要）
pip install ultralytics opencv-python numpy openvino  # openvino は任意（IR出力したい場合）
python train_golfclubhead_yolov8.py \
  --data_root /path/to/datasets/club_head \
  --imgsz 1280 \
  --model yolov8n.pt \
  --epochs 200 \
  --batch 16 \
  --device 0         # GPU:0 / "cpu"


フォルダ構成（例）：

/path/to/datasets/club_head/
  images/
    train/*.jpg|png
    val/*.jpg|png
  labels/
    train/*.txt      # YOLO txt: class x_center y_center w h (normalized)
    val/*.txt


クラス名は club_head を 0 番に置く前提（1クラス）。複数にしたい場合は YAML 内 names: を調整。


python train_golfclubhead_yolov8.pyython train_golfclubhead_yolov8.py --data_root club_head --export_ov
